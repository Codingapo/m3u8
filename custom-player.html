How to Use

Direct HLS Streaming:

This player uses HLS.js to directly play m3u8 streams without ads or external iframe embeds.

Usage:

  1. Paste your m3u8 URL in the input field
  2. Click "Load Stream" to start playback
  3. Use the quality selector for different resolutions
  4. Enjoy ad-free streaming!

Features:

  • No ads or external embeds
  • Automatic quality selection
  • Manual quality control
  • Fullscreen support
  • Mobile responsive
  • CORS proxy support

Supported Formats:

  • HLS (.m3u8)
  • MP4 (direct)
  • WebM (direct)
