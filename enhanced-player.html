Enhanced Features

Proxy Support:

Enable proxy mode to bypass CORS restrictions and access streams that require special headers.

How to Deploy Proxy:

  1. Deploy the provided proxy server to Vercel
  2. Update the proxy URL in settings
  3. Enable proxy mode
  4. Load streams that were previously blocked

Direct Mode:

When proxy is disabled, streams are loaded directly using HLS.js. This works for most public streams.

Custom Headers:

Add custom headers like Referer, User-Agent, or authorization tokens in JSON format.

Supported Formats:

  • HLS (.m3u8) - with or without proxy
  • MP4 (direct)
  • WebM (direct)
  • DASH (experimental)

Deployment Guide:

1. Upload proxy files to Vercel
2. Get your proxy URL
3. Update proxy settings
4. Enjoy ad-free streaming!
